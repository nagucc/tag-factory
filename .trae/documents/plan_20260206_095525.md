# 版本1.5 标签查询与分析功能开发计划

## 一、开发目标
实现标签查询与分析功能，支持按标签筛选数据对象、多标签组合查询、标签使用统计分析、关联关系分析和结果导出。

## 二、功能范围

### 2.1 标签查询功能
- [ ] 按单个标签查询关联的数据对象和记录
- [ ] 多标签组合查询（AND/OR/NOT逻辑）
- [ ] 标签关联关系查询
- [ ] 高级查询条件（按时间范围、操作人等）

### 2.2 标签统计分析
- [ ] 标签使用次数统计（按时间维度）
- [ ] 标签分布分析（按数据对象、标签类型）
- [ ] 标签应用趋势分析
- [ ] 标签共现关系分析

### 2.3 查询结果导出
- [ ] 导出为CSV格式
- [ ] 导出为Excel格式
- [ ] 自定义导出字段

### 2.4 前端界面
- [ ] 标签查询页面（QueryForm组件）
- [ ] 标签分析仪表盘（图表展示）
- [ ] 关联关系可视化

## 三、技术方案

### 3.1 后端API设计
```
POST /api/tags/query          # 标签组合查询
GET  /api/tags/stats/usage     # 使用统计
GET  /api/tags/stats/distribution # 分布分析
GET  /api/tags/stats/trends    # 趋势分析
GET  /api/tags/stats/co-occurrence # 共现分析
POST /api/tags/export          # 导出查询结果
```

### 3.2 数据库扩展
- 新增标签查询日志表（可选）
- 新增分析结果缓存表（可选）
- 优化标签应用表的索引

### 3.3 前端组件
- QueryForm组件（组合查询条件）
- StatsCards组件（统计卡片）
- Charts图表组件（趋势图、分布图）
- DataGrid组件（查询结果展示）
